Detailed Description

The Connection class is used to configure Iot-Ticket server connection.

Connection is a singleton. Authentication credentials are set via setUserName() and setPassword(). Usually there is no need to touch the serverUrl.

Example:

iot::Connection* connection = iot::Connection::instance();
connection->setUserName("foo");
connection->setPassword("bar);
// Now connection is configured and ready to be used.
