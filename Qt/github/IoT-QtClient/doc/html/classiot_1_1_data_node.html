Detailed Description

The DataNode class can be used to read/write device data from/to Iot-Ticket server.

DataNode can read or write one patch of values at a time. DataNode is registered to the Iot-Ticket server on the first time values are written.

bool iot::DataNode::writeCachedValues ( )

Write values in the internal buffer to the Iot-Ticket server.

This is async operation. writeFinished(bool) will be emitted once operation is done. DataNode needs to be configured before values can be written via setDeviceId(), setPath() and setName(). DataNode is created on the first time values are written to the IotTicket server. So on the first write unit should be set too.

Returns
True if writing starts (datanode is ready to write isReadyToWrite())
iot::DataNode node;
node.setDeviceId("deviceId");
node.setPath("engine";
node.setName("fanspeed");
node.setUnit("rpm");
node->writeValue(1, true);
node->writeValue(2, true);
// Values in cache (1 and 2) are send to the IotTicket server
node->writeCachedValues();
