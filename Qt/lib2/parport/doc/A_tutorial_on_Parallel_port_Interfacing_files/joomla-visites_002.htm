Veľké poďakovanie patrí tiež vám všetkým, ktorí ste sa podelili o svoju kultúru tým, že ste prispeli prekladom Joomla-Visites:

Program Joomla-Visites vrátane dokumentácie vytvoril Arnaud Lecus.

Statistics consist of seven parts : Návštevy, Zobrazené str., Pohyb, Lokalita, Prostredie, Odkazovače, Summary, available for the period Deň, Týždeň, Mesiac, Year

Joomla-Visites can for example gives these informations : Štatistiky, Návštevy, Unikátni návštevníci, Zobrazené stránky, Stránok na návštevníka, Pages per visit, Pages per significant visit, Priemerná dĺžka návštevy, Priemerný čas zobrazenia stránky, Podiel návštev jednej stránky, Average visits per day

Joomla-Visites has a powerful administration tool. For example you can :

  • Môžete vylúčiť administrátora zo štatistík. Tato metóda je založená na cookies a bude fungovať iba s prehliadačom, ktorý práve používate. Nastavenie môžete kedykoľvek zmeniť.
  • Don't count IP/IP ranges in the statistics
  • Record ALL URL variables / Do NOT record any URL variable / Record ONLY specified variables / Record all EXCEPT specified variables
  • Add newsletter / Add partner / Add site
  • Send statistics by email / Statistics in a RSS feed
  • Server Information
